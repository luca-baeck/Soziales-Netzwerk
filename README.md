# hiddlestick

a brand new social network

## Requirements
* MariaDB v 10.7.3
* libgd
* php v 7.4

### php-modules (package)
* gd (php7-gd)
* header_module
* fileinfo (php7-fileinfo)

`names of the packages are those on opensuse`

### php.ini settings
`file_uploads = On`
